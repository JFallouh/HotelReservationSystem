# HotelReservationSystem
The Web application allows potential customers to reserve rooms based on availability. 
Back-end Side (Laravel) .
Front-end Side: Use HTML, CSS and Bootstrap. 
Web API call:public Web API to provide weather forecast information for the location of the hotel.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
This is a Laravel web application configured to run with Docker and XAMPP. This project uses Docker for containerization and XAMPP for local development.

Prerequisites
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Docker and Docker Compose installed on your system.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
XAMPP installed if you plan to use XAMPP for local development.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Composer installed for managing PHP dependencies.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Node.js and npm installed for front-end dependencies (if applicable).
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Setup with Docker
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Clone the Repository
------------------------------------------------------------------------------------------------------------------------------------------------------------------
git clone https://github.com/yourusername/your-repository.git
------------------------------------------------------------------------------------------------------------------------------------------------------------------
cd your-repository
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Build Docker Images
------------------------------------------------------------------------------------------------------------------------------------------------------------------
docker-compose build
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Start Docker Containers
------------------------------------------------------------------------------------------------------------------------------------------------------------------
docker-compose up -d
------------------------------------------------------------------------------------------------------------------------------------------------------------------
The application will be available at http://localhost:8080
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Run Migrations and Seeders
------------------------------------------------------------------------------------------------------------------------------------------------------------------
docker-compose exec app php artisan migrate
------------------------------------------------------------------------------------------------------------------------------------------------------------------
docker-compose exec app php artisan db:seed
------------------------------------------------------------------------------------------------------------------------------------------------------------------

Configuration

Apache Server Name Warning
------------------------------------------------------------------------------------------------------------------------------------------------------------------

You might see a warning about the server's fully qualified domain name. This can be suppressed by setting the ServerName directive in the Apache configuration file.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
echo "ServerName localhost" | sudo tee -a /etc/apache2/apache2.conf
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Restart Apache to apply changes:
------------------------------------------------------------------------------------------------------------------------------------------------------------------

sudo service apache2 restart
